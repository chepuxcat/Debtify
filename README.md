Debtify - ваш личный финансовый дневник

легкий, open source проект, позволяющий контролировать ваши расходы и доходы.

запуск(если вы программст):
1. скачивание зависимостей 
pip install -r requirements.txt 

2. запуск
на windows: python main.py
на linux: python3 main.py
